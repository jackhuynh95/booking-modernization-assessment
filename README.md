# booking-modernization-assessment

PhoenixDX `.NET Backend Engineer` technical design assessment.

This repository is documentation-first and PDF-only. It captures a proposed booking microservice extraction from a legacy platform, including service boundaries, API contract, events, idempotency, eventual consistency, observability, edge cases, AI usage, and final submission outline.

No source-code implementation is started in this foundation phase.
